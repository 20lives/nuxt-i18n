## Development

- Fork the project and clone it in your existing Nuxt project:

```sh
cd my-nuxt-project/
mkdir modules/
git clone git@github.com:<username>/nuxt-i18n.git modules/nuxt-i18n
```

- Edit your Nuxt config to use the local module:

```js
// nuxt.config.js

{
  modules: [
    ['./modules/nuxt-i18n', {
      // options...
    }]
  ]
}
```

- Code, code, code!
- Submit awesome PRs
